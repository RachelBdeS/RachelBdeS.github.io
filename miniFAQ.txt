Rediger ces trucs dans ma FAQ
------------------------------------------------------------
• <strong></strong> remplace <b></b>
• <section></section> remplace <div>
• Couleurs en hexadecimal 
  - Du plus foncé (0) au plus clair (F): #RVB
  - Du plus foncé (00) au plus clair (FF): #RRVVBB
• NB: écrire les CSS dans des fichiers externes pour que le navigateur les chargent qu'une fois, indépendament des pages web (fichiers html, php, asp, etc.). 

• JS: foreach { <instruction>; } devient for (<variable_en_declaration> of <tableau>) { <instruction>; }
• const <variable> // est readonly
• let <variable> // a une portée limitée aux { dans lesquelles elle se trouve}
• var <variable> // a une portée limitée à la fonction dans lesquelle elle se trouve

• document.querySelector("#id") <=> document.getElementById("id")
• document.querySelector("tag") <=> document.getElementsByTagName("tag")[0]   
• var tableau_d'éléments = <dom_element>.querySelectorAll("type 1 d'élements, type 2 d'élement, etc.")
• Syntax avancée : 
  <dom_element>.querySelector (
  "#identifiant" OU ".<classe>" OU "<balise>" OU <balise>[<propriété>='<valeur>'] OU "<element> <element_enfant>" );
  Exemple: Affichera en marron tous les passage en italique dans les paragraphes: 
  var mon_element = document.querySelector("p");
  mon_element.querySelector("i").color="brown");
  
• CSS: Priorité des styles en cas de conflit : #element > .element > tag
• CSS: element {
  display: flex;
  justify-content: center; Centre verticalement son contenu (si display=flex)
  align-items: center;Centre horizontalement son contenu (si display=flex)
  }
  
• DOM acess (in css and querySelector and etc.) : 
  element + element
  element > premier_enfant_seulement
  element enfant1 + enfant2
  element enfant1 enfant_de_enfant1
  element[champs]
  element[champs='valeur_du_champs']
  
• placeholder = <TAG valeur_par_defaut_souhaitée="valeur affichée comme background dans le champs seulement quand il est vide"></TAG>

• console.log("Hello") = alert("Hello") qui ne bloque pas l'execution du code et s'affiche  dans les outils de developpement de Google Chrome.
